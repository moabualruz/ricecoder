# Suggested Commands

## Building and Testing
- `cargo build`: Build the project
- `cargo test -p ricecoder-continuous-improvement`: Run tests for this crate
- `cargo test -p ricecoder-continuous-improvement --features test-utils`: Run tests with test utilities enabled

## Code Quality
- `cargo fmt`: Format code
- `cargo clippy`: Lint code for issues

## Running
- No specific run command as this is a library crate, but can be tested with `cargo test`

## Utility Commands (Windows)
- `git status`: Check git status
- `git add .`: Stage changes
- `git commit -m "message"`: Commit changes
- `dir`: List directory contents (equivalent to ls)
- `cd <dir>`: Change directory
- `findstr <pattern> <file>`: Search in files (equivalent to grep)
- `where <command>`: Find executable location
