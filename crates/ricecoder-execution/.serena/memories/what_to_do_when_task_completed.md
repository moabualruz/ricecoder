# What to Do When a Task is Completed

After completing any code changes or additions:

1. **Format the code**: Run `cargo fmt` to ensure consistent formatting
2. **Lint the code**: Run `cargo clippy` to check for potential issues and improvements
3. **Run tests**: Execute `cargo test -p ricecoder-execution` to ensure all tests pass
4. **Update documentation**: If APIs changed, update README or docs
5. **Check CHANGELOG**: Update CHANGELOG.md if changes are significant

These steps ensure code quality, correctness, and maintainability.
