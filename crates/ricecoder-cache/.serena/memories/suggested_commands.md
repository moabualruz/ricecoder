# Suggested Commands for RiceCoder Cache Development

## Testing
- `cargo test` - Run all tests to ensure functionality works correctly

## Benchmarking
- `cargo bench` - Run performance benchmarks to measure cache performance

## Code Quality
- `cargo fmt` - Format the code according to Rust standards
- `cargo clippy` - Run the linter to check for code issues and improvements

## Building
- `cargo build` - Build the project in debug mode
- `cargo build --release` - Build the project in release mode for production

## Documentation
- `cargo doc --open` - Generate and open the project documentation

## Utility Commands (Windows)
- `dir` - List files in directory (equivalent to `ls` on Unix)
- `cd <path>` - Change directory
- `type <file>` - Display file contents (equivalent to `cat`)
- `findstr <pattern> <file>` - Search for patterns in files (equivalent to `grep`)
- `git <command>` - Version control operations
