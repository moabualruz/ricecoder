# Suggested Commands

## Build and Run
- `cargo build`: Build the project
- `cargo run --bin ricecoder-beta`: Run the CLI application
- `cargo install --path .`: Install the binary globally

## Testing
- `cargo test`: Run all tests
- `cargo test -- --nocapture`: Run tests with output visible

## Code Quality
- `cargo fmt`: Format code
- `cargo clippy`: Lint code for issues
- `cargo check`: Check for compilation errors without building

## Documentation
- `cargo doc --open`: Generate and open documentation

## System Utilities (Windows)
- `dir`: List files (equivalent to `ls`)
- `cd <dir>`: Change directory
- `type <file>`: Display file contents (equivalent to `cat`)
- `findstr <pattern> <file>`: Search in file (equivalent to `grep`)
- `dir /s <pattern>`: Find files (equivalent to `find`)
- `git <command>`: Version control

## Project-Specific
- `ricecoder-beta --help`: Show CLI help
- `ricecoder-beta run --output-dir beta-reports`: Run full beta testing program
