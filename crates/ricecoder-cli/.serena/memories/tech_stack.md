The project is built using the Rust programming language. It uses Cargo as the build system and package manager. Key technologies include:

- Tokio for asynchronous runtime
- Clap for CLI argument parsing
- Serde for serialization
- Various other crates for HTTP, file I/O, etc.
- Supports AI providers like OpenAI, Anthropic, Ollama
- LSP integration for language server protocol
- TUI interface for terminal UI
