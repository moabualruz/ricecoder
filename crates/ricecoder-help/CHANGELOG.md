# Changelog

All notable changes to the `ricecoder-help` crate will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial implementation of help system
- `HelpDialog` widget with scrollable content
- Search functionality with Ctrl+F
- Organized help by categories
- Keyboard navigation support
- Default RiceCoder help content
- Support for custom help content
- Search highlighting and result navigation
- Comprehensive test coverage

### Features
- Help dialog widget with ratatui integration
- Search functionality with real-time filtering
- Category-based organization
- Keyboard shortcuts for navigation
- Scrollable content with scrollbar
- Search result highlighting
- Case-sensitive/insensitive search options

- 10.1: Help dialog display
- 10.2: Navigation and search support
